## **關於模型**

- 參數量: **0B**
- 目前可回應句子: **11句**
- 回應語言: **中文 + 英文**

## **編譯所需環境**

- 編譯器:**`gcc`** / **`clang`** *(默認gcc)*
- 所需庫: **`stdio`**/**`stdlib`**/**`string`**/**`unistd`**/**`time`**
- 但我已經打包好發行版了! *(僅windows X86-64和Linux X86-64)*

## **Makefile** *(僅限Linux)*

- **`make`**: 編譯Linux優化後的版本
- **`make Linuxdebug`**: 編譯Linux未優化的版本
- **`make win`**: 編譯Windows優化後的版本
- **`make windebug`**: 編譯Windows未優化的版本
